## AF基础知识

### 什么是对焦（Focus）
- **对焦**：调整镜头与传感器之间的距离，使被摄物体在传感器上形成清晰的像
- **景深（DOF）**：对焦前后能够清晰成像的范围
- **焦平面**：成像最清晰的平面

### 为什么需要自动对焦（AF）
- 手动对焦速度慢、精度依赖经验
- 手机摄像头模组小，景深较浅，需要精确对焦
- 动态场景需要实时跟踪对焦

## AF系统组成

### 1. 硬件组成
| 组件 | 功能说明 |
|------|---------|
| VCM（音圈马达） | 驱动镜头移动实现对焦 |
| 霍尔传感器 | 检测镜头位置，提供闭环反馈 |
| 驱动IC | 控制VCM电流，精确控制镜头位置 |
| 图像传感器 | 捕获图像，提供对焦评价数据 |
