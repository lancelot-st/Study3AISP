## ISP概述

### 什么是ISP？
ISP（Image Signal Processor，图像信号处理器）是手机相机系统中专门用于处理图像信号的处理器。它负责将CMOS/CCD传感器输出的原始RAW数据转换为高质量的YUV/JPEG图像。

### 为什么需要ISP？
- 传感器输出的原始数据存在各种缺陷（噪声、畸变、色彩偏差等）
- 需要进行色彩空间转换和图像增强
- 实现各种拍照功能（HDR、夜景、人像等）
