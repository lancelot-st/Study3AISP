## AWB基础概念

### 什么是AWB
AWB（Auto White Balance，自动白平衡）是ISP（图像信号处理器）中的核心模块之一，用于消除不同光源下的色偏，使白色物体在任何光照条件下都呈现为白色。

### 色温与光源
| 光源类型 | 色温范围（K） | 特征 |
|---------|------------|------|
| 烛光 | 1500-2000 | 偏红/橙 |
| 白炽灯 | 2500-3500 | 偏黄 |
| 荧光灯 | 4000-5000 | 偏绿 |
| 日光 | 5500-6500 | 标准白色 |
| 阴天 | 6500-7500 | 偏蓝 |
| 阴影 | 7500-8500 | 明显偏蓝 |

### 核心参数
- **R/G, B/G比值**：描述当前场景的色温特征
- **CCT（Correlated Color Temperature）**：相关色温
- **Tint**：色调偏移量
